# 后端服务

## 安装依赖

```bash
npm install
```

## 启动服务

**方式一：使用 npm 命令**
```bash
npm start
```

**方式二：使用启动脚本（Linux/Mac）**
```bash
./start.sh
```

**方式三：使用启动脚本（Windows）**
```bash
start.bat
```

服务器将在 `http://localhost:5990` 启动

## 数据库

数据库文件会自动创建在 `backend/database.db`

## 环境变量

- `PORT`: 服务器端口（默认: 5990）
